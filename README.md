# Lecture 7 – The Code You Never Write - Made by **Muhammad Usama Zia**

This repository contains four AI-assisted Python projects demonstrating prompt engineering, verification, documentation, and workflow automation.

## Projects

### Project 1 – Money Detective
Analyze transaction history and detect subscriptions, recurring charges, duplicates, and spending categories.

### Project 2 – What's My Grade Really?
Calculate weighted grades, determine current standing, and estimate required final exam scores.

### Project 3 – The Books Don't Match
Perform financial reconciliation and identify unpaid, partially paid, overpaid, and underpaid records.

### Project 4 – Organize The Mess
Automatically categorize and organize files based on type and extension.
