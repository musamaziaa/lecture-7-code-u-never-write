# Project 4 — Organize The Mess

## Problem

People often accumulate large numbers of files in a single folder, making it difficult to locate important documents.

This project automatically categorizes files based on their extensions.

---

## AI Tool Used

Claude Sonnet

---

## Files Included

* organize_files.py
* sample_files.csv
* prompts.md
* screenshots/

---

## Objective

Categorize files into:

* Images
* Documents
* Videos
* Music
* Archives
* Other

Calculate:

* file counts
* category totals

---

## Verification

Known files:

report.pdf → Documents

essay.docx → Documents

cat.jpg → Images

vacation.png → Images

music.mp3 → Music

movie.mp4 → Videos

archive.zip → Archives

Verification passed.

---

## Findings

The script successfully categorized files according to their extensions.

Unknown extensions were placed into the Other category.

---

## Challenges

Some files may contain unsupported or missing extensions.

These were safely handled by the script.

---

## Learning

AI can automate repetitive file organization tasks effectively.

Human review is still useful for unusual file types or ambiguous extensions.
